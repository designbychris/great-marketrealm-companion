# Great Marketrealm Companion

The official companion plugin for the Great Marketrealm campaign setting.

The plugin provides a complete player and Dungeon Master toolkit including:

- Character Builder
- Character Sheets
- Campaign Management
- Equipment
- Magic Items
- Custom Races
- Custom Classes
- Journals
- Inventory
- Spell Tracking
- Party Management
